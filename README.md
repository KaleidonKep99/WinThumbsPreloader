# WinThumbsPreloader
Thumbnails preloader for Windows Explorer

![Screenshot](https://raw.githubusercontent.com/bruhov/WinThumbsPreloader/master/Website/images/preview.gif)

# CLI Options
-s : Silent mode
-r : Recursive directory search
-m : Multi-threaded generation
